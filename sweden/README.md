# Sweden Public Pack

Public Sweden pack for `Swedish Politics Data`.

Current public-safe scope:

- country: `Sweden`
- election type: `Riksdag`
- municipality election years: `2014`, `2018`, `2022`
- national trend years: `2002`, `2006`, `2010`, `2014`, `2018`, `2022`
- public geography: `municipality`
- factors: `population`, `age65`, `education`, `income`, `turnout`, `population density`, `cars`

Current next-factor priority is tracked in:

- `sweden-factor-board.md`

This pack is meant to hold the Sweden-only public scope:

- one declared election family
- one declared public geography
- one declared factor layer
- one declared national-trends layer

It should stay narrower than the broader internal source tree on purpose.
